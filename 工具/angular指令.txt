全局安装：cnpm install -g @angular/cli
本地安装：npm install --save-dev @angular/cli
全局卸载：npm uninstall -g @angular/cli
在项目路径下卸载：npm uninstall --save-dev @angular/cli
清理：npm cache verify
跨域启动：npm start
切换端口： ng serve --port xxxx
新建项目：ng new 项目名
新建组件：ng generate component 名称
zorro安装npm install ng-zorro-antd
打包：ng build --prod --aot
